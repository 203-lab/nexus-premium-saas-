# Nexus Premium SaaS Landing Page

A premium, fully responsive SaaS landing page built with pure HTML, CSS, and Vanilla JavaScript. 

## Features
- **Premium Aesthetics:** High-end glassmorphism, gradient borders, and sleek modern typography.
- **Dark Mode:** Seamless light/dark theme toggling with user preference saved in Local Storage.
- **Performance Optimized:** No heavy frameworks (No Bootstrap/Tailwind/jQuery). 100% Vanilla code. Deferred JS loading.
- **Advanced Animations:** Custom built Scroll-Reveal IntersectionObserver engine (`.reveal-up`, `.reveal-left`, etc.).
- **Micro-interactions:** Material button ripple effects, custom interactive cursor, and smooth 60fps hover states.
- **Fully Responsive:** Mobile-first CSS Grid and Flexbox implementation.
- **SEO & Accessibility:** ARIA-labels, screen-reader optimized, semantic HTML5 structure, and full OpenGraph meta tags.

## Tech Stack
- HTML5
- CSS3 (Variables, Grid, Flexbox, Animations)
- Vanilla JavaScript (DOM manipulation, Local Storage, IntersectionObserver)

## Installation & Usage
1. Clone the repository: `git clone https://github.com/203-lab/nexus-premium-saas-.git`
2. Open `index.html` in your browser. No build steps or package installations required!

## Project Structure
- `/css/style.css`: Contains all design tokens, variables, and responsive layout styling.
- `/js/script.js`: Handles preloader, dark mode logic, scroll animations, and interactive cursor.
- `/images/`: Contains optimized placeholder images and favicons.
- `index.html`: The main structured HTML document.
